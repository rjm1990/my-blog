## 📚 程序员英语的特点

### 词汇特点

- 专业术语多
- 缩写多

```
Program: 程序 vs 计划
Class: 类 vs 班级
Object: 对象 vs 物体
```

### 句子特点

- 被动语态多
- 长句子多
- 技术描述精确

## 📖 阅读技巧

### 1. 不要逐字翻译

**错误方式**
```
The function returns a promise that resolves to an array.
↓
这个 功能 返回 一个 承诺 那个 解决 到 一个 数组
```

**正确方式**
```
理解意思：这个函数返回一个Promise，resolve后是数组
```

### 2. 抓关键词

```
The map() method creates a new array 
with the results of calling a provided function 
on every element in the calling array.
```

关键词：map, creates, new array, function, every element

理解：map()方法对数组每个元素调用函数，返回新数组

### 3. 利用代码示例

文档看不懂，看代码：

```javascript
// 从代码理解功能
const arr = [1, 2, 3];
const doubled = arr.map(x => x * 2);
// doubled = [2, 4, 6]
```

### 4. 善用工具

**翻译工具**
- DeepL：翻译质量高
- Google Translate：通用
- 划词翻译插件

**词典工具**
- 有道词典
- 欧路词典

**注意**
- 不要过度依赖翻译
- 专业术语要记

## 💡 学习方法

### 每天积累

```
1. 阅读一篇英文技术文章
2. 记录不会的单词
3. 不翻译，尝试理解
```

### 看文档技巧

```
1. 先看Overview
2. 再看Quick Start
3. 不懂的地方看代码
4. 记录常用表达
```

### 建立词汇本

```markdown
# 技术词汇本

## 2026-03-20

### polyfill
- 含义：兼容旧浏览器的代码
- 例句：This feature needs a polyfill for IE11.

### memoization
- 含义：缓存计算结果
- 例句：Use memoization to optimize performance.
```

## ⚠️ 常见误区

1. **追求完美发音**: 重点是沟通，不是发音
2. **不敢开口**: 错了也没关系
3. **只背单词**: 要结合语境
4. **放弃中文**: 技术概念用英文，其他用中文
